Load Honeybee Objects Use this component to load Honeybee objects from a file on your system. The valid files are created by dump Honeybee objects component. - 

#### Inputs
* ##### filePath [Required]
A valid path to a file on your drive (e.g. c:\ladybug\20ZonesExample.HB)
* ##### load [Required]
Set to True to load the objects from the file

#### Outputs
* ##### readMe!
...
* ##### HBObjects
Script variable radOpaqueMaterial
