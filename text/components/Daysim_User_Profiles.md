#### Inputs
* ##### lightingControl [Default]
0 > Passive, 1 > active
* ##### blindControl [Default]
0 > Passive, 1 > active, 3 > based on daylight glare probability
* ##### frequency [Default]
Frequency of the year that this user type will use the space.

#### Outputs
* ##### userProfile
Daysim user profile
