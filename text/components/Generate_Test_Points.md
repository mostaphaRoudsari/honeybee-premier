#### Inputs
* ##### testSurface [Required]
Test surface as a Brep.
* ##### gridSize [Required]
Size of the test grid.
* ##### distBaseSrf [Required]
Distance from base surface.
* ##### moveTestMesh [Optional]
Set to 'False' if you want test mesh not to move. Default is 'True'.
