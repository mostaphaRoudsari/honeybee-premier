Set Exposure for HDR - 

#### Inputs
* ##### HDRFilePath [Required]
Path to an HDR image file
* ##### exposure [Default]
A number between 0 and 1
* ##### render [Required]
Set to True to render the new image

#### Outputs
* ##### outputFilePath
Path to the result HDR file
